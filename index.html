Arduino Buzzer + Light Project

Description: For my Arduino project I designed an alarm. When buzzed the alarm will flash a red light. The idea for this was I wanted something simple and neat. The task was to either do Spaceship Interface, Light Sensor, Buzzer, or Traffic Light and add your own feature to it. I decided to do the buzzer and add a red light to it when it buzzes.

Design and Production Process:
  • Brainstormed ideas
  • Watched Tutorials
  • Gathered materials
  • Tested out different wires, etc
  • Entered in my code for the Arduino

Reflection: After completing this project I learned, How to make the lights flicker according to the buzzer. One thing I would do differently is add more lights.
